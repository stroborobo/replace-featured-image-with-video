A simple plugin to replace the featured image in your WordPress post or page with a video.

== Description ==
Replace Featured Image with Video is a simple plugin to replace the featured image in your WordPress post or page with a video. Supports Youtube, Dailymotion, Vimeo and many other sites that provide embed code.. just don't forget to edit the embed code to the size of your featured image 

This plugin will attempt to override any existing featured image so removal or deletion should not be necessary. Note: You must assign a featured image to your post in order for it to be replaced (Otherwise neither image nor video will be displayed).

Tested up to 3.4.1

#### Functionality
* Replace the featured image in your WordPress website with a video.
* Supports: Youtube, Dailymotion, Vimeo and any site t hat provides embed code...


== Installation ==
Follow the steps below to install the plugin.

1. Upload the .zip file to your WordPress website
2. Activate the plugin through the 'Plugins' menu in WordPress
3. There will now be a box above the featured image option in both posts and pages - edit the embed code height and width to the size of your featured image.
